Görli

warning

The Lido testnet on Goerli will no longer be supported after February 29th, 2024. If you have (w)stETH to withdraw, please do so before this date. Moving forward, all testnet activities will shift to Holešky, where a new full-fledged Lido testnet is already operational.

See the following announcement: https://x.com/LidoFinance/status/1758458433303351784
